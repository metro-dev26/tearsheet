# Tearsheet

*Automated first-pass diligence on a company's SEC filings — every red flag traced to the exact line it came from.*

A working slice of automated investment diligence. Given a public company's SEC
filings, the system extracts financials, raises evidence-backed red flags, and
links every claim to the exact source span it relies on.

Scope of v1: a single company, a small set of genuine transaction-advisory
checks, and a grounding guard that refuses any claim it cannot trace to the
filing. The emphasis is reliability over surface area.

**Status:** in development.
